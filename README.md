# SuperTux Milestone 1 samples
The samples used in SuperTux Milestone 1 songs. There are some samples from the .mod versions of the SuperTux Milestone 2 songs too.

If you want the soundfont used for SuperTux Milestone 2, use this: https://musical-artifacts.com/artifacts/633

forest2.mod uses the same samples as forest.mod so those samples are in the forest folder.

The forest_old samples are in the old folder of the forest folder.

Salcon is not included yet.
